For first time pushing codes in github:

git init
git add .
git commit -m " Your message while pushing the codes in github"
git branch -m main
git remote add origin [your github link]
git push -u origin main


Regular code pushing:

git init
git add .
git commit -m " Your message while pushing the codes in github"
git branch -m main
git push origin main
